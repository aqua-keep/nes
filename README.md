# NES 模拟器

[English](README_en.md)

这是一个基于正点原子 NES 模拟器的项目。原项目的 CPU 使用 ARM 汇编编写，无法移植到其他平台。本项目使用 C 语言重写了 CPU 部分，
**旨在方便移植到各种嵌入式设备**（如 STM32、ESP32 等）。

## 支持的 Mapper 列表

0, 1, 2, 3, 4, 6, 7, 8, 11, 13, 15, 16, 17, 18, 19, 21, 22, 23, 24, 25, 32, 33, 34, 64, 65, 66, 67, 69, 70, 71, 72, 73,
75, 76, 78, 79, 85, 87, 88, 99, 113, 189, 225, 227, 240, 245



## 游戏键位

| 按键      | 功能     |
|---------|--------|
| W       | 上      |
| S       | 下      |
| A       | 左      |
| D       | 右      |
| ← (方向键) | A 按钮   |
| → (方向键) | B 按钮   |
| H       | Select |
| 空格      | Start  |

## 项目结构

```
nes/
├── nes_cpu.c/h     - CPU 核心 (6502)
├── nes_ppu.c/h     - 图形处理单元
├── nes_apu.c/h     - 音频处理单元
├── nes_mapper.c/h  - Mapper 支持
├── nes_main.c/h    - 主逻辑
└── mapper/         - 各 Mapper 实现
```

## 运行测试

将 NES ROM 文件放入 rom 目录，运行程序加载即可。
